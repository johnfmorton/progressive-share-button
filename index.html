Progressive Share Button Demo Page

The Progressive Share Button is a web component that will only be displayed if the browser supports the Web Share API. If the browser does not support the Web Share API, the button will not be displayed. This is a progressive enhancement, which means that the button will only be displayed if the browser supports the Web Share API. If the browser does not support the Web Share API, the button will not be displayed.

The most basic usage is using no attributes. This makes the component share the URL of the current page.

<progressive-share-button />

You can specify the URL and the title as demonstrated in the next example.

<progressive-share-button url="https://github.com/johnfmorton/progressive-share-button" title="Visit Example.com" debug=0 />

The following is the same as the basic example above, but with debug mode enabled. Debug mode will display the button even if the browser does not support the Web Share API. (If you are on Chrome on a Mac, this will be the only instance of the share button you will see on this page.) Clicking the share icon will not activate the share dialog box but instead will console.log the data that will be shared.

<progressive-share-button url="https://github.com/johnfmorton/progressive-share-button" title="Visit Example.com" class="example-0" debug=1 />

The following uses the 'smart-share' feature which will concatenate the title, text and URL into a single string and then share that string. This helps deal with some inconsistent implementations in the Web Share API.

<progressive-share-button title="Progressive Share Button" text="Check out this cool web component that creates a share button that will only be displayed if the browser supports the Web Share API." url="https://github.com/johnfmorton/progressive-share-button" smart-share=1 icon-size="20" debug=0 class="example-1" />

The following examples demonstrate overriding the icon built into the component.
